# report — 风险全局视图

> **职责**：/pace-guard report 子命令的详细执行规则。**自包含**——不依赖 guard-procedures-common.md（只读分析，不需要自主权矩阵）。

## 数据源

- `.devpace/risks/` 所有风险文件

## 输出格式

```markdown
## 风险报告

### 严重度分布
| 等级 | 数量 | 占比 |
|------|------|------|

### Open 风险列表（按严重度降序）
| 编号 | 来源 | 严重度 | 关联 CR | 发现摘要 | 建议下一步 |
|------|------|--------|--------|---------|-----------|

### 已处理风险（本迭代）
| 编号 | 最终状态 | 处理方式 |
|------|---------|---------|
```

## 分层输出

| 层级 | 输出 |
|------|------|
| 简要（`--brief`） | `report：{{open数}} open（H:{{H}} M:{{M}} L:{{L}}）· 本迭代解决 {{R}} · [趋势方向]` |
| 标准（默认） | 上方完整格式 |
| 详细（`--detail`） | 完整格式 + 按 PF 分组视图 + 总体风险评分 + 建议优先处理列表 |

## 渐进暴露

`.devpace/risks/` 不存在时输出"暂无风险数据，使用 `/pace-guard scan` 执行首次风险扫描。"
