# 默认输出规程

> **职责**：/pace-next 默认模式的输出格式规程。由 SKILL.md 路由表指定加载。

## 输出格式（≤ 3 行）

```
💡 建议：[行动建议]
   原因：[1 句推理]——[≤15 字推理后缀]
   操作：[对应命令或自然语言指引]（[1 句行为预览]）
```

### 推理后缀规则

在"原因"行末尾附加 ≤15 字的推理依据，如"审批是推进阻塞点（3 个 CR 被阻塞）"。显而易见时可省略。

### 操作行增强

命令引导附带 1 句行为预览，如"→ /pace-review（查看摘要并确认合并）"。
