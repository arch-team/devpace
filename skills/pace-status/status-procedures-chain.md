# chain：价值链定位

> 由 SKILL.md 路由表加载。仅在 `chain` 子命令时读取。

chain 是 `trace <当前活跃 CR 所属 PF>` 的特化版——聚焦"你在这里"的定位，使用用户友好术语（不使用 BR/PF/CR）。

## 步骤

1. 读取 `.devpace/state.md` 当前工作 + `.devpace/project.md` 价值功能树
2. 定位当前活跃 CR 在价值链中的位置
3. 以树形视图输出

## 输出示例

```
目标: 实现用户认证系统
  └── 功能: 登录模块 (🔄)
        ├── 任务: 表单验证 (✅)
        └── 任务: OAuth 集成 (🔄) ← 你在这里
```

无活跃 CR 时显示最近完成的链路位置。

## 导航

输出末尾追加 1 行：`→ 追溯其他功能：/pace-status trace [名称]`
