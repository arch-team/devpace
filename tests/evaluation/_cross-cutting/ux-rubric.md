# UX 原则评分表

> 基于 design.md §2 的 UX 原则 P1-P7，每项 1-3 分，总分 /21。

## 评分标准

| 分数 | 定义 |
|------|------|
| **3** | 完全满足：行为自然、体验流畅、无可改进 |
| **2** | 基本满足：核心正确，有小瑕疵或偶尔不一致 |
| **1** | 不满足：原则被明显违反或未实现 |
