# 场景 S11: 自由探索

**对应需求**: S11, NF8
**验证目标**: 默认探索模式、不触发流程、可切换

## 前置条件

- 已完成 S01（.devpace/ 已初始化）

## 执行步骤

| # | 人类动作 | 期望 Claude 行为 |
|---|---------|-----------------|
| 1 | 说"看看 [文件/模块]" | 自由分析，不修改 .devpace/ 文件 |
| 2 | 说"分析一下 [问题]" | 提供分析，不创建 CR |
| 3 | 说"评估 [方案]" | 评估利弊，不触发状态机 |
| 4 | 说"帮我改 [功能]" | 切换到推进模式（绑定 CR） |
| 5 | 说"还是先不改了" | 退回探索模式 |

## 验收标准

- [ ] 默认进入探索模式，无需显式声明
- [ ] 探索过程中不创建 CR、不触发质量检查、不修改 .devpace/ 文件
- [ ] 可随时通过修改代码的意图切换到推进模式
- [ ] 可退回探索模式

## 评分维度

| 维度 | Pass | Partial | Fail |
|------|------|---------|------|
| 默认探索 | 无需声明即为探索 | 需显式声明 | 默认推进 |
| 状态隔离 | .devpace 无任何修改 | 偶尔修改 | 频繁修改 |
| 模式切换 | 自然切换 | 需显式命令 | 无法切换 |

## 检查产物

- `.devpace/` — git diff 确认探索期间无修改
